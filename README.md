# NHL Draft ML Project

## Objetivo
Prever se um jogador draftado terá mais de 100 jogos na NHL.

## Tipo de problema
Classificação binária.

## Target
- 1 = mais de 100 jogos
- 0 = 100 jogos ou menos

## Etapas
1. Carregar os dados
2. Explorar o dataset
3. Criar variável alvo
4. Selecionar features
5. Tratar dados faltantes e categóricos
6. Treinar modelo
7. Avaliar desempenho
